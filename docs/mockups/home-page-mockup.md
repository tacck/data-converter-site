## レスポンシブ対応

### デスクトップ (1024px以上)

- 機能カードを2カラムで横並び表示
- カード間の余白: 32px
- 最大幅: 1200px (中央揃え)

### タブレット (768px-1023px)

- 機能カードを2カラムで横並び維持
- カード間の余白: 24px
- 余白を調整

### モバイル (767px以下)

- 機能カードを1カラムで縦積み
- カード間の余白: 16px
- 全幅表示(左右に16pxの余白)

- すべてのカードにキーボードフォーカス対応
- カードリンクに適切な `aria-label` を設定
- 十分なカラーコントラスト確保

- セマンティックHTML使用 (`<nav>`, `<main>`, `<section>`)

## インタラクション

1. ユーザーがトップページにアクセス
2. 機能カードが表示される
3. カードにホバーすると視覚的フィードバック(シャドウ強調)
4. カードまたはCTAボタンをクリック
5. 対応する機能ページに遷移

## アニメーション

- ページロード時: カードがフェードイン(0.3秒)
- ホバー時: シャドウとスケールの変化(0.2秒)
