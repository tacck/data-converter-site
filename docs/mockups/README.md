# 画面モックアップ

Data Converter Siteの各ページの画面モックアップドキュメントです。

## モックアップ一覧

### [トップページ](./home-page-mockup.md)

- URL: `/`
- 概要: サイトの入り口となるページ。各機能へのナビゲーションを提供
- 主要要素:
  - ヒーローセクション
  - 機能カード(日時変換、カラーコード変換)
  - 今後の追加予定機能

### [日時変換ページ](./datetime-page-mockup.md)

- URL: `/datetime`
- 概要: 日時とUnix Timeの相互変換機能を提供
- 主要要素:
  - 日時 → Unix Time 変換セクション
  - Unix Time → 日時 変換セクション
  - タイムゾーン選択
  - 単位選択(秒/ミリ秒)
  - 日時形式選択(標準/ISO)

### [カラーコード変換ページ](./color-page-mockup.md)

- URL: `/color`
- 概要: 複数のカラーフォーマット間の相互変換機能を提供
- 主要要素:
  - カラープレビュー
  - カラー形式タブ(RGB, ARGB, HEX, CMYK, HSL)
  - 入力フォーム
  - 変換結果一覧
  - CSS形式コピー機能

## 共通デザイン要素

### ナビゲーションバー

すべてのページで共通のナビゲーションバーを使用:

- サイト名: "Data Converter Site"
- リンク: [ホーム] [日時] [カラー]
- 現在のページをハイライト表示

### カラースキーム

- プライマリカラー: #3B82F6 (青)
- 背景色: #F9FAFB (薄いグレー)
- カード背景: #FFFFFF (白)
- テキスト: #111827 (ダークグレー)

### タイポグラフィ

- フォントファミリー: Geist Sans, sans-serif
- 見出し: 600-700 (Semi-Bold to Bold)
- 本文: 400 (Regular)

### レスポンシブブレークポイント

- デスクトップ: 1024px以上
- タブレット: 768px-1023px
- モバイル: 767px以下

## デザイン原則

### シンプルさ

- 不要な装飾を排除
- 機能に集中したUI
- 明確な視覚的階層

### 使いやすさ

- 直感的な操作フロー
- リアルタイムフィードバック
- 明確なエラーメッセージ

### アクセシビリティ

- WCAG 2.1 AA準拠を目指す
- キーボード操作対応
- スクリーンリーダー対応
- 十分なカラーコントラスト

### レスポンシブデザイン

- すべてのデバイスで快適に使用可能
- モバイルファースト設計
- タッチ操作に最適化

## 実装時の注意点

### コンポーネント化

- 再利用可能なコンポーネントを作成
- 入力フィールド、ボタン、カードなどを共通化
- Tailwind CSSのユーティリティクラスを活用

### 状態管理

- React Hooksを使用
- 各ページで独立した状態管理
- グローバル状態管理ライブラリは不要

### バリデーション

- リアルタイムバリデーション実装
- ユーザーフレンドリーなエラーメッセージ
- 入力フィールドの視覚的フィードバック

### パフォーマンス

- 変換処理は100ms以内
- 不要な再レンダリングを防止
- React.memo、useCallback、useMemoを活用
