## コンポーネント詳細

### ナビゲーションバー

- サイト名: "Data Converter Site"
- リンク: [ホーム] [日時] [カラー]
- 現在のページ(日時)はハイライト表示

### 日時 → Unix Time セクション

#### 入力要素

1. **日時形式選択** (ラジオボタン)
   - 標準 (YYYY/mm/DD HH:MM:SS)
   - ISO 8601

2. **タイムゾーン選択** (ドロップダウン)
   - オプション: UTC, Asia/Tokyo, America/New_York, Europe/London など

3. **日時入力フィールド** (テキスト入力)
   - プレースホルダー: "2024/01/01 12:00:00" または "2024-01-01T12:00:00Z"
   - バリデーション: リアルタイムで形式チェック
   - エラー表示: 入力フィールド下に赤文字で表示

4. **単位選択** (ラジオボタン)
   - 秒
   - ミリ秒

5. **Unix Time表示フィールド** (読み取り専用テキスト)
   - 変換結果を表示
   - コピーボタン付き

6. **変換ボタン**
   - ラベル: "変換する"
   - プライマリカラー

### Unix Time → 日時 セクション

#### 入力要素

1. **単位選択** (ラジオボタン)
   - 秒
   - ミリ秒

2. **Unix Time入力フィールド** (数値入力)
   - プレースホルダー: "1704110400"
   - バリデーション: 数値のみ受け付け
   - エラー表示: 入力フィールド下に赤文字で表示

3. **タイムゾーン選択** (ドロップダウン)
   - オプション: UTC, Asia/Tokyo, America/New_York, Europe/London など

4. **日時形式選択** (ラジオボタン)
   - 標準 (YYYY/mm/DD HH:MM:SS)
   - ISO 8601

5. **日時表示フィールド** (読み取り専用テキスト)
   - 変換結果を表示
   - コピーボタン付き

6. **変換ボタン**
   - ラベル: "変換する"
   - プライマリカラー

## エラー状態の表示例

```
日時を入力:
┌─────────────────────────────────────────────┐
│ 2024/13/01 12:00:00                        │ ← 赤枠で強調
└─────────────────────────────────────────────┘
⚠ 無効な日時形式です。YYYY/mm/DD HH:MM:SS形式で入力してください。
```

## レスポンシブ対応

### デスクトップ (1024px以上)

- 2カラムレイアウト: 日時→Unix Time と Unix Time→日時 を横並び

### タブレット (768px-1023px)

- 2カラムレイアウト維持、余白を調整

### モバイル (767px以下)

- 1カラムレイアウト: 縦積み
- 入力フィールドを画面幅いっぱいに表示

## アクセシビリティ

- すべての入力フィールドに `<label>` を関連付け
- エラーメッセージに `role="alert"` と `aria-live="polite"` を設定
- エラー状態の入力フィールドに `aria-invalid="true"` を設定
- キーボードナビゲーション対応
- フォーカス状態を明確に表示

## インタラクション

1. ユーザーが日時を入力
2. リアルタイムでバリデーション実行
3. 「変換する」ボタンをクリック
4. Unix Timeが表示フィールドに表示される
5. コピーボタンでクリップボードにコピー可能
