## コンポーネント詳細

### ナビゲーションバー

- サイト名: "Data Converter Site"
- リンク: [ホーム] [日時] [カラー]
- 現在のページ(カラー)はハイライト表示

### カラープレビューセクション

- 大きな正方形または長方形のプレビューエリア
- 入力された色をリアルタイムで表示
- 中央にHEXコードを表示
- 背景色が入力された色に変化

### カラー形式選択タブ

- タブ形式で切り替え: RGB, ARGB, HEX, CMYK, HSL
- アクティブなタブは下線で強調

### RGB入力フォーム

#### 入力要素

1. **R (Red) 入力フィールド**
   - 数値入力 (0-255)
   - ラベル: "R:"
   - 範囲表示: "(0-255)"

2. **G (Green) 入力フィールド**
   - 数値入力 (0-255)
   - ラベル: "G:"
   - 範囲表示: "(0-255)"

3. **B (Blue) 入力フィールド**
   - 数値入力 (0-255)
   - ラベル: "B:"
   - 範囲表示: "(0-255)"

4. **変換ボタン**
   - ラベル: "変換する"
   - プライマリカラー

### ARGB入力フォーム

```
ARGB値を入力:

A: ┌──────┐  R: ┌──────┐  G: ┌──────┐  B: ┌──────┐
   │ 255  │     │ 255  │     │  87  │     │  51  │
   └──────┘     └──────┘     └──────┘     └──────┘
   (0-255)      (0-255)      (0-255)      (0-255)
```

### HEX入力フォーム

```
HEX値を入力:

┌─────────────────────────────────────────────┐
│ #FF5733                                     │
└─────────────────────────────────────────────┘
形式: #RRGGBB または #AARRGGBB
```

### CMYK入力フォーム

```
CMYK値を入力:

C: ┌──────┐  M: ┌──────┐  Y: ┌──────┐  K: ┌──────┐
   │   0  │     │  66  │     │  80  │     │   0  │
   └──────┘     └──────┘     └──────┘     └──────┘
   (0-100%)     (0-100%)     (0-100%)     (0-100%)
```

### HSL入力フォーム

```
HSL値を入力:

H: ┌──────┐  S: ┌──────┐  L: ┌──────┐
   │  11  │     │ 100  │     │  60  │
   └──────┘     └──────┘     └──────┘
   (0-360)      (0-100%)     (0-100%)
```

### 変換結果セクション

#### 表示要素

- すべてのカラー形式の変換結果を一覧表示
- 各形式の横に「コピー」ボタン
- CSS形式でのコピーボタン群
  - RGB形式: `rgb(255, 87, 51)`
  - RGBA形式: `rgba(255, 87, 51, 1.0)`
  - HEX形式: `#FF5733`
  - HSL形式: `hsl(11, 100%, 60%)`

## レスポンシブ対応

### デスクトップ (1024px以上)

- プレビューと入力フォームを横並び
- 変換結果は下部に全幅表示

### タブレット (768px-1023px)

- プレビューと入力フォームを縦積み
- 入力フィールドは横並び維持

### モバイル (767px以下)

- すべて縦積みレイアウト
- 入力フィールドも縦積み
- タブは横スクロール可能

## アクセシビリティ

- すべての入力フィールドに `<label>` を関連付け
- カラープレビューに `aria-label` で色情報を提供
- エラーメッセージに `role="alert"` と `aria-live="polite"` を設定
- 成功メッセージに `role="status"` を設定
- エラー状態の入力フィールドに `aria-invalid="true"` を設定
- キーボードナビゲーション対応
- タブ切り替えをキーボードで操作可能
- 十分なカラーコントラスト確保

## インタラクション

1. ユーザーがカラー形式タブを選択
2. 選択した形式の入力フォームが表示される
3. カラー値を入力
4. リアルタイムでプレビューが更新
5. 「変換する」ボタンをクリック
6. すべての形式の変換結果が表示される
7. 各形式の「コピー」ボタンでクリップボードにコピー
8. CSS形式ボタンでCSS用の文字列をコピー
9. 成功メッセージが表示される

## カラープレビューの詳細

- サイズ: 200x200px (デスクトップ)、150x150px (モバイル)
- 角丸: 8px
- ボーダー: 1px solid #e0e0e0
- 中央にHEXコード表示
- HEXコードの文字色は背景色に応じて自動調整(白または黒)
