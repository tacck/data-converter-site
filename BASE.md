# Data Converter Site

## 目的

シンプルな、データ変換機能を提供するWebアプリの構築。

## 機能

下記機能を保持する。

機能ごとにページ(URLパス)があり、今後の追加も容易に可能。

- 日時
  - 一般的な日時とUnix Time(エポックタイム)の相互変換
  - 日時は `YYYY/mm/DD HH:MM:SS` 形式で表示・入力
  - 日時はISO形式の表示・入力にも対応
  - 日時はタイムゾーンに対応
  - Unix Time は、秒・ミリ秒どちらも利用可能
- カラーコード
  - RGB形式で表示・入力可能
  - RGBは数値・HEXどちらも利用可能
  - ARGBも上記と同様に利用可能
  - CMYKも利用可能
  - HSL形式も利用可能
  - 入力された色はリアルタイムでサンプル表示
  - CSSとしてコピー可能

## 開発

### ドキュメント

アーキテクチャドキュメント、詳細実装ドキュメント、を生成・更新しながら開発を行なう。
ドキュメントは Markdown 形式で、図は可能な限り Mermaid を利用する。
作成したドキュメントは docs ディレクトリに配置する。

#### アーキテクチャドキュメント

下記を含める。アーキテクチャの理解に必要であれば、それ以外も記載する。

- プロジェクト概要
- 利用技術
- 実装機能概要
- インフラ構成
- サイトマップ
- クラス図

#### 詳細実装ドキュメント

機能ごとに詳細なドキュメントを生成する。
そのため、機能ごとにディレクトリを作成し、その中にドキュメントを作成する。

下記の中で、必須のもの以外は必要なものを記載する。

- 機能概要 (必須)
- フローチャート
- シーケンス図
- 状態遷移図

### テスト

テスト駆動開発を行なう。

- クラスやコンポーネントは、単体テストを実装する。
- 統合テストとして、Playwright を使用し、E2Eテストを行なう。
